# SimpleTimer:

This is a simple timer class in swift that enables you to:
- Local scoped timer
- Chainable
- One liners
- Use regular callbacks

## Install

```
Copy paste /*SPM support comming soon*/
```

## Usage

```
SimpleTimer(interval: 3,repeats: true){print("tick")}.start()//Ticks every 3 seconds
```
